# Testing hydrodynamic response to initial-state geometry in Pb+d^↑ collisions

## eprint: [link](https://arxiv.org/abs/2509.00511)

This folder contains parameter files to compute observables in Pb+d^↑
collisions as proposed in the paper, arXiv:2509.00511.
